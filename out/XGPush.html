registerPush(aliasopt) → {external:Promise}

与信鸽服务注册。注册的时候可以填别名，按照别名发送时，只有经过注册的别名才会收到推送。 可以重复调用这个方法。如果变更别名，则替换原有注册的结果。

 设备的别名

// 直接注册
xgpush.registerPush().then(function() {

  console.log('oh no: ' + errCode);
});

// 带设备别名注册
xgpush.registerPush('foo').then(function() {
  console.log('success');

}).catch(function(errCode) {
  console.log('oh no: ' + errCode);
});
